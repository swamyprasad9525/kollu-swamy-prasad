## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization

- Update personal details in `src/components/Hero.jsx`, `Skills.jsx`, `Projects.jsx`, etc.
- Modify colors/theme in `src/index.css`.
